Many of FretPet's functions can be accessed through keyboard shortcuts. Some of these shortcuts may only be available when the appropriate palette is open.

Toolbar

spacebar Start or stop Playing the document.
esc Stop Playing.
A Add (Duplicate) Chord.
F Toggle Follow View.
E Toggle Edit Mode.
L Toggle Looping.
T Toggle Transform Mode.
' Toggle Solo Mode.

Chords / Document

↑ ↓ Move the document cursor, selecting the Current Chord.
⇧↑ ⇧↓ Move the document cursor, extending the selection.
⌃↑ ⌃↓ Move the document cursor by pages.
⌃⇧↑ ⌃⇧↓ Move the document cursor by pages, extending the selection.
Page Up, Page Down, Home, End Change the position of the Document view.
← → Select the next / previous part.
⌫ or ⌦ Delete the selected chords.
A Add a copy of the Current Chord.

Scale Palette

⌥↑ ⌥↓ ⌥← ⌥→ Move the Scale Cursor.
⌥ Tab Add the Scale Tone to the Current Chord and move the Scale Cursor up a third.

Guitar Palette

[  ] Move the Fret Bracket
B Toggle the Fret Bracket
H Toggle Horizontal View
N Toggle Numbered Dots
U Toggle Unused Tones
